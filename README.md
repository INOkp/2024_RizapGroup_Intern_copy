# Intern_25

## 命名規則

### issue名

- 【HTML】 `What`を`How`<br>

  （例）  【HTML】 sectionタグにトップ画像を追加

- 【CSS】 `What`を`How`<br>

  （例）  【CSS】 h2.classの色を赤色に変更

### ブランチ名

- issues/`イシュー番号`<br>

  （例）  issues/3

### コミット名

- #`イシュー番号` `What`を`How`<br>

  （例）  #3 sectionタグにトップ画像を追加

### プルリクエスト名

- [`ブランチ名`] `What`を`How`<br>

  （例）  [issues/3] sectionタグにトップ画像を追加
